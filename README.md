# ⚡ SiteBuilder AI
      
      Полнофункциональный конструктор сайтов с 93 компонентами и AI генерацией.
      
      ## 🚀 Демо
      **[Открыть конструктор](https://sasaburk4659-coder.github.io/site-builder-ai)**
      
      ## 🎯 Возможности
      - 🎨 **93 компонента** — навигация, hero, контент, формы, e-commerce, и т.д.
      - 🤖 **Telegram Bot** — конструктор ботов с готовым кодом
      - 🧠 **AI / Нейросети** — генератор текста, изображений, чат-бот
      - 💬 **Мессенджеры** — WhatsApp, Telegram, Viber, Messenger
      - 💳 **Платежи** — кнопки оплаты, корзина, калькулятор
      - 📈 **Аналитика** — дашборд, счётчики, heatmap
      - 🗺️ **Интеграции** — карты, календарь, погода, соцсети

      ## 📦 Использование
      1. Откройте `index.html` в браузере
      2. Перетащите компоненты из левой панели
      3. Или нажмите 🤖 Сгенерировать и опишите сайт
      4. Экспортируйте готовый HTML
      
      ## 📋 Все 93 компонента
      **Навигация (5):** Header, Sticky Navbar, Sidebar, Breadcrumbs, Mega Menu
      **Hero (5):** CTA, Video, Slider, Form, Fullscreen
      **Контент (10):** Text, Heading, Rich Text, Columns, Image+Text, Quote, Divider, Spacer, Code, Markdown
      **Секции (8):** Features Grid/List, About, Team, Stats, Timeline, Steps, Partners
      **Коммерция (6):** Pricing, Product Card/Grid, CTA, Offer, Cart
      **Соцдоказательство (4):** Testimonials, Reviews, Logos, Rating
      **Медиа (7):** Gallery, Video, Slider, Lightbox, Before/After, Icon Box, Audio
      **Формы (6):** Contact, Subscribe, Login, Register, Search, Multi-step
      **Данные (5):** Table, FAQ, Tabs, Progress, Charts
      **Telegram Bot (6):** Commands, Inline KB, Reply KB, Messages, Webhook, Menu
      **AI (7):** Text Gen, Image Gen, Chatbot, Rewriter, SEO, Translate, Form Fill
      **Мессенджеры (5):** WhatsApp, Telegram, Viber, Messenger, Live Chat
      **Платежи (4):** Payment, Donation, Calculator, Order
      **Аналитика (3):** Dashboard, Visitors, Heatmap
      **Интеграции (5):** Map, Social, Embed, Calendar, Weather
      **Footer (3):** Simple, Columns, CTA
      **Декор (4):** Gradient, Parallax, Counter, Particles
